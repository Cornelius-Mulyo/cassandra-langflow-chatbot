# Student Database Context

This file provides context for a chatbot built using Langflow.

The Cassandra database stores student records including:
- Student ID
- Name
- Major
- GPA

Example students:

Cornelius Mulyokela – Computer Science – GPA 3.89  
Alice Johnson – Data Science – GPA 3.75  
Michael Lee – Cybersecurity – GPA 3.92

The chatbot can answer questions about students, majors, and GPA information.
